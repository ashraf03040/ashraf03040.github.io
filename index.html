ScanDump Learning Lab (Zybo)

I’m building a hands-on lab to learn scan dump capture + analysis using a Zybo Z7 (Zynq-7000). This complements my background in test program development and silicon debug workflows.

Scan dump DFT / Scan Vivado + Vitis Python analysis

Project goals

  • Drive scan control signals (scan_clk / scan_en / scan_in) from PL/PS.
  • Capture scan_out stream and store it (UART dump).
  • Parse + align captured data in Python and generate summary plots.
